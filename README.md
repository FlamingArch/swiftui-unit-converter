# Unit Converter
A simple Unit Converter written using SwiftUI. Converts lengths, volumes, temperatures and timings.

## Features
- Contains examples on how to use `TextFields`, `Segmented Pickers`, etc.
- Implemention of SwiftUI `Form`, `TabView`, Sidebar using `List` and `NavigationView`,.
- TabView in Phones, Sidebar in Large Screen, providing custom tailored experiences for iPhone and iPad.
- Shows how SwiftUI `@State` One-way and Two-way bindings works.
